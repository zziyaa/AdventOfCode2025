# Advent of Code 2025

My solutions for [Advent of Code 2025](https://adventofcode.com/2025), written in Go.

Each day's solution lives in its own directory. Run a solution by passing the puzzle input through standard input:

```sh
go run ./day01 < input.txt
```

Puzzle solutions in this repository are implemented by me without AI assistance. Once an implementation produced the correct answer, I generally did not optimize it further, so the solutions may not be the most elegant ones available.
